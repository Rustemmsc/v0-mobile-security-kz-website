## Вход в систему

### Доступ к админ-панели

- **URL**: `https://cctv.mobilesecurity.kz/admin/auth/login`
- Введите ваш email и пароль
- После успешного входа вы попадете на дашборд

### Первый вход

Если администратор еще не создан:
1. Перейдите по адресу `/admin/setup`
2. Введите email и пароль для нового администратора
3. Вы автоматически войдете в систему

## Управление категориями

### Зачем нужны категории

Категории помогают организовать товары и упрощают навигацию на сайте для покупателей. Перед добавлением товаров рекомендуется создать все необходимые категории.

### Создание категории

1. Перейдите в раздел **"Категории"** в боковом меню
2. Нажмите кнопку **"Добавить категорию"**
3. Заполните форму:

#### Обязательные поля:
- **Название (RU)** - Например: "IP камеры"
- **Название (KK)** - Например: "IP камералар"
- **Название (EN)** - Например: "IP Cameras"

#### Дополнительные поля:
- **Описание (RU/KK/EN)** - Краткое описание категории для SEO
- **Иконка** - Имя иконки из библиотеки Lucide (например: `camera`, `shield`, `lock`)
- **Порядок отображения** - Число для сортировки (меньше = выше в списке)

4. Нажмите **"Создать"**

### Редактирование категории

1. В таблице категорий найдите нужную
2. Нажмите кнопку с иконкой редактирования (карандаш)
3. Измените необходимые поля
4. Нажмите **"Обновить"**

### Изменение порядка категорий

Используйте стрелки вверх/вниз в таблице для изменения порядка отображения категорий на сайте.

### Скрытие/удаление категории

- **Показать/Скрыть**: Используйте переключатель в столбце "Видимость"
- **Удаление**: Нажмите кнопку корзины (внимание: все товары этой категории потеряют связь)

## Управление товарами

### Создание товара

1. Перейдите в раздел **"Товары"** в боковом меню
2. Нажмите кнопку **"Добавить товар"**
3. Заполните форму на трех языках:

#### Основная информация (на трех языках):

**Русский язык:**
- **Название товара (RU)*** - Обязательно. Например: "IP камера Hikvision DS-2CD2143G0-I 4MP"
- **Описание (RU)** - Подробное описание товара для покупателей

**Қазақ тілі:**
- **Тауар атауы (KK)*** - Обязательно. Например: "IP камера Hikvision DS-2CD2143G0-I 4MP"
- **Сипаттамасы (KK)** - Толық сипаттама

**English:**
- **Product Name (EN)*** - Required. Example: "Hikvision DS-2CD2143G0-I 4MP IP Camera"
- **Description (EN)** - Full product description

#### Дополнительные параметры:

- **Категория** - Выберите категорию из списка (должна быть создана заранее)
- **Цена*** - Обязательно. Укажите цену в тенге (например: 45000)
- **Артикул (SKU)** - Уникальный код товара для учета (например: "HIKV-DS2CD2143")
- **Количество на складе** - Текущий остаток товара (например: 15)
- **Бренд** - Производитель (например: "Hikvision")
- **Модель** - Модель товара (например: "DS-2CD2143G0-I")

#### Изображения товара:

1. Нажмите на область загрузки изображений
2. Выберите файлы с компьютера (рекомендуется JPG или PNG)
3. Подождите загрузки
4. Можно загрузить несколько изображений
5. Первое изображение будет главным

**Рекомендации по изображениям:**
- Размер: минимум 800x800px
- Формат: JPG или PNG
- Качество: четкие фото на белом фоне
- Количество: 3-5 фотографий товара с разных ракурсов

4. Нажмите **"Создать товар"**

### Редактирование товара

1. В таблице товаров найдите нужный
2. Нажмите кнопку с иконкой редактирования
3. Измените необходимые поля
4. Нажмите **"Обновить товар"**

### Быстрые действия с товарами

В таблице товаров доступны:
- **Показать/Скрыть** - Переключатель видимости на сайте
- **Редактировать** - Открывает форму редактирования
- **Удалить** - Полное удаление товара (будьте осторожны!)

### Статус товара

- **В наличии** - Товар доступен для заказа
- **Нет в наличии** - Товар временно недоступен
- **Активен/Скрыт** - Отображается ли товар на сайте

## Управление контентом сайта

Раздел "Контент сайта" позволяет управлять всеми текстами на сайте без изменения кода.

### Создание элемента контента

1. Перейдите в раздел **"Контент сайта"**
2. Нажмите **"Добавить контент"**
3. Заполните форму:

#### Технические поля:

- **Ключ*** - Уникальный идентификатор (например: `hero.title`, `about.description`)
  - Используйте точку для группировки: `раздел.название`
  - Только латинские буквы, цифры, точки и подчеркивания
  
- **Раздел** - Название раздела для группировки (например: `hero`, `about`, `contact`)

- **Тип контента** - Выберите формат:
  - **Текст** - Обычный текст
  - **HTML** - HTML-разметка
  - **Markdown** - Текст в формате Markdown
  - **Изображение URL** - Ссылка на изображение
  - **URL** - Обычная ссылка

- **Описание** - Пояснение для чего используется этот контент

#### Значения на трех языках:

- **Значение (RU)** - Текст на русском языке
- **Мәні (KK)** - Қазақ тіліндегі мәтін
- **Value (EN)** - Text in English

### Примеры использования

**Заголовок главной страницы:**
\`\`\`
Ключ: hero.title
Раздел: hero
Тип: text
Описание: Главный заголовок на главной странице

Значение (RU): Профессиональные системы видеонаблюдения
Мәні (KK): Кәсіби бейнебақылау жүйелері
Value (EN): Professional Video Surveillance Systems
\`\`\`

**Текст "О компании":**
\`\`\`
Ключ: about.description
Раздел: about
Тип: html
Описание: Описание компании на странице "О нас"

Значение (RU): <p>Мы предоставляем <strong>комплексные решения</strong> для безопасности...</p>
\`\`\`

### Редактирование контента

1. Найдите элемент в таблице
2. Нажмите кнопку редактирования
3. Измените текст на нужных языках
4. Нажмите **"Обновить"**

## Управление администраторами

**Доступно только для Super Admin**

### Роли администраторов

Система поддерживает 4 уровня доступа:

1. **Super Admin** - Полный доступ ко всему, включая управление администраторами
2. **Admin** - Управление товарами, категориями, заказами и контентом
3. **Manager** - Управление товарами и просмотр заказов
4. **Editor** - Только редактирование контента сайта

### Создание нового администратора

1. Перейдите в раздел **"Администраторы"** (видно только Super Admin)
2. Нажмите **"Добавить администратора"**
3. Заполните форму:
   - **Email** - Рабочий email нового администратора
   - **Полное имя** - ФИО администратора
   - **Пароль** - Временный пароль (минимум 8 символов)
   - **Роль** - Выберите уровень доступа
4. Нажмите **"Создать администратора"**
5. **ВАЖНО**: Скопируйте и передайте пароль новому администратору
6. Попросите администратора сменить пароль при первом входе

### Изменение роли администратора

1. Найдите администратора в таблице
2. Нажмите кнопку **"Изменить роль"**
3. Выберите новую роль
4. Нажмите **"Обновить"**

### Удаление администратора

1. Найдите администратора в таблице
2. Нажмите кнопку удаления (корзина)
3. Подтвердите действие

**Внимание**: Нельзя удалить самого себя или последнего Super Admin.

## Резервное копирование

### Зачем нужны бэкапы

Резервные копии защищают ваши данные от случайного удаления или изменения. Рекомендуется создавать бэкап перед:
- Массовым изменением товаров
- Обновлением цен
- Удалением большого количества данных

### Создание резервной копии

1. Перейдите в **"Настройки"**
2. В разделе **"База данных"** нажмите **"Создать резервную копию"**
3. Подождите завершения процесса
4. Появится сообщение об успешном создании бэкапа

**Что сохраняется в бэкапе:**
- Все товары
- Все категории
- Весь контент сайта
- Все заказы

**Что НЕ сохраняется:**
- Учетные записи администраторов
- Настройки системы

### Восстановление из резервной копии

1. В разделе **"База данных"** найдите список бэкапов
2. Выберите нужную резервную копию по дате и времени
3. Нажмите **"Восстановить"**
4. Подтвердите действие в диалоговом окне
5. Дождитесь завершения восстановления

**ВНИМАНИЕ**: 
- Восстановление заменит все текущие данные на данные из бэкапа
- Это действие необратимо
- Рекомендуется создать бэкап текущего состояния перед восстановлением

### Автоматическое удаление старых бэкапов

Бэкапы старше 7 дней автоматически удаляются для экономии места в базе данных.

## Полезные советы

### Порядок работы при запуске магазина

1. **Создайте категории** - Сначала все категории товаров
2. **Настройте контент** - Тексты главной страницы, "О нас", контакты
3. **Добавьте товары** - По категориям, с качественными фото
4. **Создайте бэкап** - После заполнения всех данных
5. **Добавьте администраторов** - Если работаете в команде

### Оптимизация для поисковых систем (SEO)

- Заполняйте описания товаров подробно и на всех трех языках
- Используйте уникальные названия товаров
- Добавляйте ключевые слова в описания категорий
- Указывайте бренд и модель для каждого товара

### Работа с изображениями

- Сжимайте изображения перед загрузкой (используйте TinyPNG или аналоги)
- Называйте файлы понятно: `hikvision-ds2cd2143-front.jpg`
- Делайте фото на нейтральном фоне
- Первая фотография - самая важная, она показывается в каталоге

### Управление остатками

- Регулярно обновляйте количество на складе
- Используйте статус "Нет в наличии" вместо удаления товара
- Скрывайте товары которые больше не продаете

### Безопасность

- Меняйте пароль каждые 3 месяца
- Не передавайте пароль по незащищенным каналам
- Выходите из системы на общих компьютерах
- Регулярно создавайте резервные копии

## Частые вопросы

**Q: Можно ли массово загрузить товары из Excel?**  
A: Пока нет, но эта функция планируется в будущих обновлениях.

**Q: Как изменить язык интерфейса админ-панели?**  
A: Сейчас интерфейс только на русском. Многоязычность интерфейса появится позже.

**Q: Что делать если забыл пароль?**  
A: Обратитесь к Super Admin для сброса пароля или используйте функцию восстановления на странице входа (если настроена).

**Q: Можно ли вернуться к предыдущей версии товара?**  
A: Да, если вы создали резервную копию перед изменением. Используйте функцию восстановления из бэкапа.

**Q: Сколько администраторов можно создать?**  
A: Неограниченное количество, но рекомендуется создавать только необходимые учетные записи.
